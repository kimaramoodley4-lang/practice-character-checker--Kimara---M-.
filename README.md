# Practice: Character Checker

A simple Node.js program that demonstrates string manipulation and user input.

## Description
This application prompts the user for a word or phrase and then asks for a specific index number. It uses JavaScript bracket notation to retrieve and display the character located at that specific position in the string.

## How to Run
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the script: `node index.js`

## Concepts Covered
* Using the `readline-sync` module for user input.
* Working with string indexing (zero-based).
* Converting string input to numbers.
